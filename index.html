Asistia Bienvenido a asistia!

Este es el sitio web de Asistia, un asistente para la gestión de asistencia y puntualidad de las empresas colombianas

Objetivo general del proyecto:

Esta aplicación digital tiene como objetivo optimizar la manera en la que se manipula la asistencia de cualquier empresa. Es una aplicación intuitiva hecha a medida para que su uso sea fácil y sencillo para cualquier persona

Lista de Funciones Principales
  • Registro digital
  • Control de puntualidad
  • Generación de reportes
