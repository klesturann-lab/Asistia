Este es el sitio web de Asistia! Un asistente para la gestión de asistencia y puntualidad de las empresas colombianas

Objetivo general del proyecto

El objetivo de esta aplicación digital, será el de optimizar la manera en la que se manipula la asistencia de cualquier empresa. Es una aplicación intuitiva hecha a medida para ser fácil y optima de usar. Lista de Funciones Principales
  • Registro digital
  • Control de puntualidad
  • Generación de reportes
